# 平台搜索指南

## Reddit 搜索

### 搜索方式

使用 web search 工具搜索 Reddit 内容，构造查询：

```
site:reddit.com [关键词] [子版块名可选]
```

### 搜索策略

1. **热帖搜索** — 搜索近期高讨论度帖子：
   ```
   site:reddit.com/r/[subreddit] [关键词] 2025
   ```

2. **讨论型搜索** — 寻找有深度的讨论帖：
   ```
   site:reddit.com [关键词] "what do you think" OR "thoughts on" OR "experience with"
   ```

3. **机会型搜索** — 寻找项目/机会相关讨论：
   ```
   site:reddit.com [关键词] "opportunity" OR "looking for" OR "built" OR "launched"
   ```

### 帖子评估标准

| 指标 | 优先级 | 说明 |
|------|--------|------|
| 评论数 > 50 | 高 | 说明讨论充分 |
| 时间 < 30天 | 高 | 保证信息新鲜 |
| Upvote > 100 | 中 | 社区认可度 |
| 有实质性评论 | 高 | 比单纯点赞更有价值 |

### 信息提取模板

从每个有价值的帖子提取：

- **标题**：原帖标题
- **链接**：原帖 URL
- **一句话摘要**：核心观点或发现
- **讨论亮点**：评论中的关键洞察
- **关联维度**：可联动到读书/行路/成事的哪个部分
